# 🚗 Car Price Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Car Price dataset using Python. The objective is to understand the factors that influence car prices and present meaningful insights through data visualization.

## 🛠️ Technologies Used

* Name
* Location
* Year
* Kilometers_Driven
* Fuel_Type
* Transmission
* Owner_Type
* Mileage
* Engine
* Power
* Colour
* Seats
* No. of Doors
* New_Price
* Price

## 📂 Dataset

The dataset contains information about different cars, including:

* Car Name
* Company
* Year
* Price
* Kilometers Driven
* Fuel Type

## 📊 Analysis Performed

* Data Cleaning
* Missing Value Handling
* Duplicate Value Checking
* Descriptive Statistics
* Company-wise Car Analysis
* Fuel Type Analysis
* Price Distribution
* Mileage Analysis
* Correlation Analysis
* Data Visualization using Matplotlib & Seaborn

## 📈 Visualizations

* Bar Chart
* Histogram
* Pie Chart
* Scatter Plot
* Line Plot
* Heatmap

## 🔍 Key Insights

* Identified the distribution of car prices.
* Compared average prices across different companies.
* Analyzed the popularity of different fuel types.
* Explored the relationship between mileage and price.
* Visualized important trends using charts.

## 🚀 Future Improvements

* Build a Car Price Prediction model using Machine Learning.
* Develop an interactive dashboard.
